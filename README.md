# 树莓派4-openwrt

使用Github Action编译树莓派4的Openwrt，感谢P3TERX的[Actions-OpenWrt](https://github.com/P3TERX/Actions-OpenWrt)项目

## 使用方法

1. 点击Fork将项目Fork到自己的仓库

2. Fork完成后进行一次新的编译：
    - 点击Star  
    - 更新.config  

3. 在Action中下载编译好的项目
